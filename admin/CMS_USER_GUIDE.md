# AXINOVA CMS 관리자 시스템 사용 설명서

## 🎯 개요

AXINOVA 평생교육원을 위한 웹 기반 관리자 시스템(CMS)입니다.  
**코딩 없이** 웹사이트의 모든 콘텐츠를 관리할 수 있습니다.

---

## 📂 시스템 구조

```
admin/
├── login.html              # 관리자 로그인
├── dashboard.html          # 대시보드 (메인)
├── content-manager.html    # 메인 페이지 관리
├── program-manager.html    # 프로그램 관리
├── blog-manager.html       # 블로그 관리
├── user-manager.html       # 회원 관리
├── media-library.html      # 미디어 라이브러리
├── settings.html           # 사이트 설정
├── css/                    # 스타일시트
│   ├── admin-login.css
│   ├── admin-dashboard.css
│   └── admin-content.css
├── js/                     # JavaScript
│   ├── admin-login.js
│   ├── admin-auth.js
│   ├── admin-dashboard.js
│   └── admin-content.js
└── data/                   # 데이터 저장소 (JSON)
```

## 📝 메인 페이지 관리

웹사이트의 메인 페이지 콘텐츠를 편집합니다.

### 편집 가능한 섹션

#### 1. 히어로 섹션
- 메인 타이틀
- 서브 타이틀
- 설명 문구
- 버튼 텍스트 & 링크 (2개)

**예시**:
```
메인 타이틀: AI 시대를 선도하는
서브 타이틀: AXINOVA 평생교육원
설명: 2026 공공 AI 자격인증 대비 실무 중심 교육
버튼1: 교육 프로그램 보기 → #programs
버튼2: 문의하기 → #contact
```

#### 2. 소개 섹션
- 섹션 제목
- 서브 제목
- 메인 설명
- 추가 설명 (선택)

#### 3. 특징 섹션
- 섹션 제목
- 특징 아이템 (추가/삭제 가능)
  - 아이콘 (Font Awesome)
  - 제목
  - 설명

**아이콘 찾기**: https://fontawesome.com/icons

**예시**:
```
아이콘: fa-graduation-cap
제목: 실무 중심 교육
설명: 현장에서 바로 활용 가능한 실무 중심의 커리큘럼
```

#### 4. 통계 섹션
- 통계 아이템 (추가/삭제 가능)
  - 숫자 (예: 500+, 95%)
  - 레이블 (예: 수강생, 만족도)

#### 5. 연락처 섹션
- 섹션 제목
- 전화번호
- 이메일
- 주소
- 운영 시간

### 저장 방법
1. 원하는 섹션을 편집
2. 상단 **"저장"** 버튼 클릭
3. "변경사항이 성공적으로 저장되었습니다!" 메시지 확인

### 미리보기
- 상단 **"미리보기"** 버튼 클릭
- 새 탭에서 메인 사이트 열림
- 변경사항 확인

## 💾 데이터 저장 방식

### 현재 (Phase 1)
- **LocalStorage**: 브라우저에 데이터 저장
- 간단하고 빠름
- 서버 불필요

### 향후 (Phase 2)
- **JSON 파일**: 서버에 JSON 형식으로 저장
- 백업 및 복원 가능
- 여러 관리자 동시 작업 가능

## 🔒 보안

### 세션 관리
- 24시간 자동 로그아웃
- "로그인 상태 유지" 시 LocalStorage 사용
- 비활성화 시 SessionStorage 사용 (브라우저 종료 시 로그아웃)

### 비밀번호 변경 (필수)
1. **설정 > 계정 관리** (개발 중)
2. 현재 비밀번호 입력
3. 새 비밀번호 입력 (8자 이상, 영문+숫자+특수문자)
4. 저장

> 💡 **Tip**: 비밀번호는 정기적으로 변경하세요!

## 🚀 워크플로우 예시

### 시나리오 1: 메인 페이지 히어로 섹션 수정

1. 관리자 페이지 로그인
2. **메인 페이지 관리** 클릭
3. **히어로 섹션** 탭 선택
4. "메인 타이틀" 수정: `AI 교육의 새로운 기준`
5. "설명 문구" 수정
6. 상단 **"저장"** 버튼 클릭
7. **"미리보기"** 버튼으로 확인
8. 메인 사이트 새로고침 → 변경사항 반영 확인

### 시나리오 2: 새 특징 추가

1. **메인 페이지 관리** > **특징** 탭
2. 하단 **"+ 특징 추가"** 버튼 클릭
3. 아이콘: `fa-award`
4. 제목: `국가 공인 자격증`
5. 설명: `2026 공공 AI 자격인증 대비 체계적 교육`
6. **"저장"** 클릭

## 🛠️ 문제 해결

### 로그인이 안 돼요
- ID와 비밀번호 확인
- 기본 계정: `admin` / `axinova2026`
- 캐시 삭제 (Ctrl+Shift+Del)
- 시크릿 모드로 시도

### 저장이 안 돼요
- 브라우저 LocalStorage 지원 확인
- 브라우저 콘솔(F12) 오류 확인
- 다른 브라우저로 시도

### 변경사항이 반영되지 않아요
- 메인 사이트 새로고침 (Ctrl+F5)
- 캐시 삭제 후 재시도
- 시크릿 모드로 확인

### 세션이 자주 끊겨요
- "로그인 상태 유지" 체크
- 쿠키 및 LocalStorage 허용 확인

## 💡 Best Practices

### 콘텐츠 작성 팁
1. **간결하게**: 사용자는 긴 텍스트를 읽지 않습니다
2. **명확하게**: 전문 용어보다 쉬운 표현 사용
3. **시각적으로**: 이미지와 아이콘 적극 활용
4. **일관성**: 톤 앤 매너 유지

### 이미지 가이드라인
- **해상도**: 최소 1920x1080 (Full HD)
- **포맷**: JPG (사진), PNG (로고/아이콘)
- **용량**: 1MB 이하 권장 (압축)
- **비율**: 16:9 (히어로), 1:1 (프로필), 4:3 (갤러리)

### SEO 최적화
- **제목**: 50-60자 이내
- **설명**: 150-160자 이내
- **키워드**: 주요 키워드 자연스럽게 포함
- **대체 텍스트**: 모든 이미지에 alt 속성
